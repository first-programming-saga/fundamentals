- `firstPythonProgram.ipynb`
    - 変数と代入: 四則演算だけでなく、余りや累乗などを含む。特に`=`の意味に注意
- `compoundOperators.ipynb`
    - 複合代入演算子: ある変数に演算を行い、元の変数に代入する場合に使う記法
- `booleanTest.ipynb`
    - 論理演算: 値の比較などの結果は`True`または`False`になる
- `quiz.ipynb`
    - 確認テスト
- `exercises.ipynb`
    - 演習問題
